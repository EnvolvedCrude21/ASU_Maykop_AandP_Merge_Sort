# ASU_Maykop_AandP_Merge_Sort

Репозиторий по дисциплине «Алгоритмизация и программирование». Здесь собраны материалы по модулю 3 и два домашних задания, подготовленные для выгрузки в один общий репозиторий.

## Состав репозитория

- `Модуль 3/merge_sort` — уже готовый проект по сортировке слиянием.
- `Homework_1_Benchmark` — доработанный бенчмарк сортировок.
- `Homework_2_BasicUT` — решение задачи на вычисление номера недели по дате с unit-тестами.

## Домашнее задание 1 — Benchmark

Что добавлено и улучшено:

- поддержаны тестовые случаи для `std::string`;
- оставлен исходный вариант `Bubble Sort`;
- добавлена оптимизированная модификация `Bubble Sort` с ранним выходом и сокращением диапазона прохода;
- добавлена сортировка вставками (`Insertion Sort`);
- результаты прогонов сохраняются в CSV-файл;
- сборка разделена на основной исполняемый файл и runner для тестов.

Ключевая идея оптимизации:

- улучшенный `Bubble Sort` завершает работу раньше, если массив уже отсортирован;
- в лучшем случае сложность уменьшается с `O(n^2)` до `O(n)`.

## Домашнее задание 2 — BasicUT

Что реализовано:

- решение через классы `Date` и `WeekNumberCalculator`;
- ввод даты в формате `гггг.мм.дд`;
- вычисление номера недели по стандарту ISO 8601;
- проверка корректности даты;
- unit-тесты с выводом имени теста в консоль.

## Сборка

### Homework_1_Benchmark


